# Minimalist— Portfolio

Personal portfolio website showcasing projects, skills, and experience as a Front-End Developer & UI/UX Designer.

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript
- Google Fonts — Inter, JetBrains Mono, Instrument Serif

## Features

- Light/dark theme toggle with OS preference detection
- Scroll reveal animations with staggered entrance
- Responsive mobile navigation with drawer overlay
- Project gallery modal with keyboard & touch navigation
- SEO meta tags & Open Graph social sharing
- Contact section with email, phone, LinkedIn, GitHub

## Sections

Hero · About · Skills · Projects · Experience · Education · Certifications · Awards · Gears · Contact

## Run Locally

Open `index.html` in your browser — no build step required.
